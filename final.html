Project Summary

This website was created as a final project to demonstrate web development skills using HTML, CSS, and JavaScript. The website focuses on GreenTech Solutions, a fictional company dedicated to sustainability and environmentally responsible technology solutions.

HTML Features

The project includes multiple webpages, navigation menus, semantic HTML elements, forms, tables, and organized page layouts.

CSS Features

CSS was used to create a consistent layout, responsive design, improved readability, and a professional visual appearance across all pages.

JavaScript Features

JavaScript was added to improve interactivity by creating alert messages and improving the overall user experience.

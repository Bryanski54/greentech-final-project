# GreenTech Solutions Final Project

This project is a multi-page website created for the final web development project. The website represents GreenTech Solutions, a fictional company focused on environmentally friendly technology and sustainability.

## Features

- Multiple webpages
- Responsive design
- Navigation menu
- Contact form
- Tables and structured content
- JavaScript interactivity
- Mobile-friendly layout
- Accessibility features

## Technologies Used

- HTML
- CSS
- JavaScript

## Pages Included

- Home
- About Us
- Products & Services
- Blog
- Contact
- Final Project

## Purpose

The purpose of this project is to demonstrate understanding of front-end web development concepts, including responsive design, accessibility, page navigation, and user interaction.
